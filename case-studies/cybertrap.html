What I Learnt

LLM Function Calling

Implemented stealth function calling with Groq Llama 3.3 70B to extract intelligence invisibly from scammer conversations. The key was making the extraction happen as a side effect of natural conversation flow.

4-Stage Extraction Funnel

Designed a sophisticated Hook ‚Üí Technical Friction ‚Üí Pivot ‚Üí Extraction funnel. The stage buffer logic prevents eager extraction and builds rapport before attempting to gather sensitive data.

Consensus Confidence System

Built a scoring system that reaches 100% confidence only when values are confirmed across multiple conversation turns. This prevents false positives and ensures extracted intelligence is reliable.

Multi-Language Support

Implemented natural language detection supporting English, Hindi, Tamil, Telugu, and Malayalam with realistic code-mixing. Mrs. Shanthi naturally switches between languages like a real South Indian grandmother would.

Conclusion

CyberTrap was a fascinating dive into AI agent design for a hackathon. Creating Mrs. Shanthi - a convincing 68-year-old retired Tamil teacher persona - taught me about prompt engineering and maintaining character consistency across conversation turns.

The combination of FastAPI for the backend and React with TypeScript for the real-time dashboard made it easy to build a complete intelligence visualization system. The dark cyber theme really brought the "honey-pot" concept to life.

This project gave me hands-on experience with LLM function calling, conversation state management, consensus-based validation, and building resilient AI agents that can handle adversarial inputs from actual scammers.

Thank you for giving your time.
