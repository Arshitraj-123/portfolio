What I Learnt

JWT Authentication Flow

Implemented secure JWT-based authentication with bcrypt password hashing. Built complete auth flow including login, register, and token-based password reset functionality with proper token expiration handling.

Browser Notifications API

Integrated browser push notifications to alert users about upcoming task deadlines. Learned about permission handling, notification scheduling, and creating a seamless user experience with deadline reminders.

Express.js REST API Design

Built a clean RESTful API with express-validator for input validation, proper error handling middleware, and CORS configuration for secure frontend-backend communication.

React + Vite Development

Leveraged Vite 7 for lightning-fast Hot Module Replacement and optimized builds. Created a smooth development experience with React 19 and React Router for seamless navigation.

Conclusion

TaskFlow was an excellent full-stack learning experience. Building secure authentication from scratch taught me the importance of proper password hashing, token management, and security best practices.

The combination of React 19 with Vite and Express.js 5 proved to be a powerful, modern stack for building responsive web applications. SQLite with Knex.js provided a lightweight yet robust database solution.

I'm particularly proud of the browser notification system that helps users stay on top of their task deadlines, and the clean, responsive UI that works beautifully across all devices.
