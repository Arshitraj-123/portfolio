What I Learnt

D3.js for Custom Charts

Building candlestick charts and custom visualizations with D3.js taught me about SVG manipulation, scales, and creating interactive data visualizations from scratch.

WebSocket Connection Management

Managing persistent WebSocket connections for real-time price feeds required careful handling of reconnection logic, heartbeats, and subscription management.

API Rate Limiting

Working with CoinGecko's free tier taught me about implementing request queues, caching strategies, and graceful degradation when rate limits are hit.

Vue 3 Composition API

Using composables for shared logic like useWebSocket, usePriceAlerts, and usePortfolio made the codebase much more maintainable and testable.

Conclusion

CryptoTracker was an exciting project that combined real-time data handling with complex data visualization. The challenge of keeping charts smooth while receiving constant price updates pushed me to optimize rendering performance carefully.

Vue 3's Composition API proved to be excellent for this type of application. The ability to extract reusable logic into composables made testing and maintenance significantly easier.

The most satisfying feature was the price alert system - seeing a notification appear exactly when a target price was hit felt magical after implementing the complex subscription logic.

Thank you for giving your time.
